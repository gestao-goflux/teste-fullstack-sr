# GoTuck

Esse projeto utiliza a api via Lumen, pois é uma api pequena, então optei por utilizar a versão mais simples do Laravel para essa criação.

# Configuração do projeto

composer install

php artisan migrate

# Subindo a api

php -S localhost:8000 -t public
